Arquitetura WEB

HTML

Esta é a primeira aula de Arquitetura Web e estamos aprendendo HTML. Uma linguagem de Marcação que representa o esqueleto de Sites Webs. Aprendizado e uso das tags, semântica, estrutura. E que, de forma prática, poderemos criar um website futuro.

CSS

Introdução à linguagem de estilização CSS, para darmos cor e alguns outros aspectos visuais à nossa estrutura HTML

Introdução ao Cliente e Servidor

Aprendemos um pouco sobre os lados do Cliente e Servidor, HTTP e HTTPS, DNS, Sites SSR e SSG, LPA. Introdução básica nos principais termos que fazem parte do sistema Frontend e sua comunicação com o Backend.
